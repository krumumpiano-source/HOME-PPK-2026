# 🏠 HOME PPK 2026

ระบบบริหารจัดการบ้านพักครูโรงเรียนพะเยาพิทยาคม

## 📋 ภาพรวม

HOME PPK 2026 เป็นระบบ Web Application สำหรับบริหารจัดการบ้านพักครูและแฟลต พัฒนาด้วย HTML/CSS/JavaScript และ Google Apps Script

## 🚀 Features หลัก

- ✅ บันทึกค่าน้ำ/ค่าไฟประจำเดือน
- ✅ แจ้งยอดชำระและตรวจสลิป
- ✅ จัดการคำร้อง (เข้าพัก/ย้าย/คืน/แจ้งซ่อม)
- ✅ ลำดับคิวรอเข้าพักอาศัย
- ✅ บัญชีรายรับรายจ่าย
- ✅ ระบบสิทธิ์การใช้งาน (Permission)
- ✅ ตั้งค่าระบบสำหรับแอดมิน
